## Evaluated but deferred

- Sophia DoH, scheduled cleanup, WSL, OneDrive, and default-app associations: deferred because this slice does not yet have reliable typed detection plus exact restore for every Windows build. No script execution substitute was added.
- WindowsFeatureProvider: deferred until feature enable/disable can be performed through a native typed API with reliable state detection and reverse operation; DISM/CMD/PowerShell wrappers are not acceptable.
- Appx removal: preview and safety classification ship, but apply is explicitly blocked because uninstall restoration cannot currently be guaranteed.
- Win11Debloat AI/Start controls beyond documented reversible registry entries: deferred pending per-build Microsoft documentation and restore tests.
